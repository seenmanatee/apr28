# Anniversary Website

A static website for a special anniversary surprise.

## Setup

1. Place your video file as `video.mp4` in the root directory.
2. Add flower images as `flower1.png`, `flower2.png`, etc.
3. Add photos as `photo1.png`, etc.
4. Add a heart icon as `heart.png`.
5. Edit the text in `index.html` to personalize the message.

## Preview

To preview in VSCode, install the "Live Server" extension and right-click `index.html` to open with Live Server.

## Hosting on GitHub.io

1. Create a new repository named `yourusername.github.io`.
2. Upload all files to the repository.
3. The site will be available at `https://yourusername.github.io`.

## Customization

- Edit `styles.css` for colors and animations.
- Modify `script.js` for more interactivity.
- Add more images and elements as needed.
